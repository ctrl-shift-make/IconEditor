Icon Editor in Small Basic
==========================

This program is a 16 x 16 plane PPM (Portable Pixmap) bitmap editor written in Microsoft Small Basic language. Following files are used to run this program.

* IconEditor.exe
* SmallBasicLibrary.dll
* img\new.png
* img\open.png
* img\save.png
* img\pencil.png
* img\pen color.png
